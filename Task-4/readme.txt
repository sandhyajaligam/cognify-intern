# Task: Interactive Webpage with JavaScript

## Objective

Create a visually styled webpage with basic JavaScript interactivity.

## Features

- A clean and colorful design using embedded CSS.
- A button that changes the background color randomly when clicked.
- Smooth transition effect for background color changes.

## Technologies Used

- HTML5
- CSS3 (Internal)
- JavaScript (Inline)

## How It Works

1. Loads a styled webpage with headings and a message.
2. Clicking the **"Change Background"** button selects a random pastel color and applies it to the background.
