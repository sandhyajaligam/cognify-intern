# Task: Dynamic Content with API Integration

## Objective

Enhance the website by using JavaScript to fetch and display data from a public API.

## Steps

1. Selected a public API (e.g., JSONPlaceholder).
2. Used `fetch()` in `script.js` to get JSON data.
3. Parsed the data and updated the DOM dynamically.

## Technologies

- HTML
- CSS
- JavaScript (fetch API)

## How to Use

- Open `index.html` in a browser.
- Data will load dynamically and appear on the page.
